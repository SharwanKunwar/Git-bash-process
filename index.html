Steps to Make a New Repository Using Git Bash Terminal

  1. Creating Git in a folder or directory by using this command: git init
  2. Navigate to your project directory using cd /path/to/your/project
  3. Stage files using git add .
  4. Commit the changes with git commit -m "Initial commit"
  5. (Optional) Add a remote repository using git remote add origin <remote-url>
  6. (Optional) Push changes to the remote repository using git push -u origin master
